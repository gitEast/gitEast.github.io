3.1 对挂载的子元素进行区分

将 render 函数根据子元素的类型，进行拆分

  • 子元素
    • HTML 元素 => 挂载标签 => mountElement

3.2 简化 vnode 的编写(h 函数)

每次写一个 虚拟 DOM，都需要编写一个 { tag: ‘’, props: ‘’, children: [] }，怪累的，写一个 h 函数简化这个过程，只需要手动输入必要的标签名、属性、子元素
